Detailed Description

Chemical synapse from a spiking neuron's axon's segment.

Synapses are the inputs of neurons and receive data from the axon of a Neuron. This class models the connection to a source Neuron::axon with physycal, chemical and genetic attributes: length, strength and prob_inheritable.

Relationships with other classes
The Synapse class is the base class of PlasticSynapse used in the vectors of synapses by type for each Neuron and there are 10 types of synapses implemented in the SpikingEvaluator class wich evauates all Neurons in a NeuralNetwork.
