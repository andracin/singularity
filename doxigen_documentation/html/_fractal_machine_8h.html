Description
Es un objeto que al ser evaluado con un dataset de entradas y salidas esperadas como "ejemplos" de entrenamiento, produce una eficiencia en la tarea de imitar las salidas esperadas para algún patrón de entradas que haya aprendido desde un ejemplo (patrón en el dataset) durante el entrenamiento.

Author

Description
Máquina de turing que en cada iteración, lee y ejecuta una celda de una cinta de instrucciones que ejecuta sobre su estado. Está compuesta por: -Un registro de estado descrito por un fractal de objetos (máquinas de turing) con conexiones jerárquicas -Una cinta compuesta de celdas de instrucciones (FractalTape.h) -Un cabezal que lee(y escribe) las celdas de instrucciones (FractalTape.h) -Una tabla de instrucciones que ejecutan los objetos del estado
